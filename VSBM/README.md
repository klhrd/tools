# WebGL Volume Shader Benchmark (VSBM)

這是一個基於 WebGL 的體積著色器（Volume Shader）基準測試工具。它利用 **光線步進 (Raymarching)** 技術即時渲染複雜的 Mandelbulb 分形，並以此測試 GPU 的浮點運算、反三角函數計算能力及像素填充率。

## 🚀 功能特點

-   **即時分形渲染**：使用純 GLSL 代碼在 GPU 上運算高度複雜的數學形狀。
-   **統一強度控制**：透過一個級數變數連動所有運算參數，簡化測試流程。
-   **URL 動態調壓**：無需修改代碼，直接透過網址後綴調整測試壓力。
-   **Live Coding**：支援即時編輯 KERNEL 函數並重新編譯，觀察不同演算法的效能差異。
-   **全平台支援**：相容於所有支援 WebGL 的桌面及行動裝置瀏覽器。

## 🎮 使用方法

### 1. 基本測試
直接打開 `vsbm.html` 即可開始標準強度測試（預設難度為 `1.0`）。

### 2. 控制測試強度（推薦）
您可以透過網址的 **Hash (#)** 功能直接指定難度級數。難度級數為正浮點數：

-   **低負載模式**：`vsbm.html#0.5`
-   **標準測試**：`vsbm.html#1.0`
-   **高壓測試**：`vsbm.html#5.0`
-   **極限壓力**：`vsbm.html#10.0`

> **注意**：當難度超過一定限制時，顯示卡可能會出現明顯卡頓（掉幀）或驅動程式重啟。

### 3. 手動設定 (CONFIG)
點擊畫面左上角的 **CONFIG** 按鈕：
-   您可以在 `textarea` 中直接修改 `kernal` 函數的 GLSL 代碼。
-   點擊 **APPLY** 即時編譯並套用新代碼。

## ⚙️ 強度連動邏輯

當調整難度級數（`difficulty`）時，程式會自動連動以下參數：

| 參數 | 說明 | 影響維度 |
| :--- | :--- | :--- |
| **Kernel Iterations** | 分形疊代的次數 | GPU 數學運算壓力 |
| **Raymarching Steps** | 光線搜尋的最大步進數 | 渲染深度與視野複雜度 |
| **Solver Steps** | 表面定位的二分搜尋次數 | 邊緣平滑度與運算精度 |

## 🖱️ 互動操作

-   **旋轉視角**：滑鼠左鍵拖拽 / 單指觸控。
-   **平移相機**：滑鼠右鍵拖拽 / 雙指觸控平移。
-   **縮放 (Zoom)**：滑鼠滾輪 / 雙指捏合。進入分形內部會顯著增加 GPU 負擔。

---

### 技術聲明
本工具僅供硬體測試與學術研究使用。高強度的測試可能會導致瀏覽器短暫無回應，請確保在執行極限測試前已保存其他工作。
