# **3D Sun Path Simulator (太陽運行路徑模擬器)**

一個基於網頁的互動式 3D 太陽運行軌跡模擬工具。透過直觀的 3D 視覺化介面，幫助使用者理解不同緯度、季節與時間下的太陽位置變化。

## **🌟 功能特色 (Features)**

* **3D 視覺化場景**：包含地景、天球網格、地軸與太陽軌跡的即時渲染。  
* **多季節軌跡**：可切換顯示夏至、春秋分、冬至的太陽運行路徑。  
* **精準定位**：  
  * 支援瀏覽器 GPS 自動定位。  
  * 提供手動輸入緯度功能。  
  * 快速緯度調整滑桿（可選）。  
* **智慧時間軸**：  
  * 具備磁吸功能的時間滑桿，可自動吸附至日出、正午、日落時刻。  
  * 即時顯示當地時間與日出日落時間。  
* **觀測數據**：  
  * 即時顯示太陽仰角 (Altitude) 與方位角 (Azimuth)。  
  * 支援切換方位角 (0-360°) 或象限角 (N/S E/W) 顯示模式。  
* **日夜環境模擬**：背景天空顏色隨太陽高度角自動變化（晨昏、白天、黑夜）。  
* **響應式設計**：針對桌面與行動裝置優化的玻璃擬態 (Glassmorphism) UI。

## **🛠️ 技術棧 (Tech Stack)**

* **HTML5 / CSS3**  
* [**Three.js**](https://threejs.org/) \- 3D 渲染引擎  
* [**Tailwind CSS**](https://tailwindcss.com/) \- 樣式框架  
* **FontAwesome** \- 圖標庫

## **⚙️ 操作說明**

* **旋轉視角**：按住滑鼠左鍵拖曳 (或單指滑動)。  
* **縮放視角**：滾動滑鼠滾輪 (或雙指縮放)。  
* **時間調整**：拖曳下方時間軸，或點擊日出/日落/正午標記快速跳轉。  
* **設定選單**：點擊右上角選單按鈕，可調整顯示圖層、定位與介面設定。
