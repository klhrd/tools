# (📆 日期倒數機 (Countdown PWA))[https://klhrd.github.io/tools/count_down/]

一個清爽、簡單、專注於日期倒數的漸進式網頁應用程式 (PWA)。它提供跨平台的使用體驗，讓您輕鬆追蹤重要的日子、考試、假期或紀念日。

---

## 🚀 使用說明

### 1. 新增與編輯活動
* 在 **「全部活動」** 頁面，點擊右下角的 **`+` 懸浮按鈕**。
* 填寫**標題**、**目標日期**等資訊。
* 可設定**重複週期**（週/月/年），用於生日或定期事件。
* 可設定**標籤**進行分類，並選擇是否**定選**至首頁。

### 2. 管理活動
* 點擊任一活動卡片，會彈出**詳細資料視窗**。
* 在詳細資料視窗中，您可以執行：
    * **定選/取消定選**：決定是否在「首頁」顯示。
    * **封存/解除封存**：手動將活動移至或移出封存區。
    * **🔔 通知**：為該活動單獨設定**每日提醒**或**特定倒數天數**（如倒數 7 天、3 天）提醒。
    * **編輯**：修改活動細節。

### 3. 首頁與篩選
* **「首頁精選」**：只顯示您已**定選**的活動，提供快速瀏覽。
* **「全部活動」**：顯示所有**活躍中**的活動。
* **標籤篩選**：點擊頁面頂部的標籤，可以過濾當前頁面顯示的活動。

### 4. 設定與清理
* 在 **「設定」** 頁面：
    * 管理**標籤**。
    * 管理**預設活動集**（例如國定節日），您可以選擇載入或隱藏預設集。
    * 設定**每日通知時間**。
    * **顯示/隱藏已封存**：切換查看已封存的活動。
    * **清理所有已封存**：永久刪除所有已封存的活動資料。
