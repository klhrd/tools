## [🔬 LensSim - 薄透鏡光線追蹤模擬器](https://klhrd.github.io/tools/LensSim/)
### **基於** **HTML**/**CSS**/**JavaScript** 的**互動式網路應用程式**
#### **用途**
* **視覺化** **薄透鏡成像公式**（**高斯公式** $\frac{1}{u} + \frac{1}{v} = \frac{1}{f}$）
* **視覺化** **主要光線追蹤圖解**
#### **使用者互動**
* 透過**滑桿**調整**焦距**（$f$）
* 透過**滑桿**調整**物距**（$u$）
* 透過**滑桿**調整**物體高度**（$H$）
* **即時觀察** 成像位置、大小、正倒立和虛實性質
### ✨ **主要功能**
* **互動式控制**
    * 透過**滑桿**即時調整**焦距** $f$（**凸透鏡**/**凹透鏡**）
    * 透過**滑桿**即時調整**物距** $u$
* **光線追蹤**
    * 繪製**三條主要光線**的路徑
    * 清楚展示**實像**或**虛像**的形成過程
* **虛像延伸線**
    * 當形成虛像時，繪製折射光線向後延伸的**粉紅色虛線**
    * **交點**即為**虛像頂端**
* **即時結果**
    * 顯示計算出的**像距** ($v$)
    * 顯示計算出的**放大率** ($M$)
    * 顯示計算出的**成像性質**（**實像**/**虛像**、**正立**/**倒立**、**放大**/**縮小**）
* **單檔案設計**
    * 整個應用程式包含在一個 **HTML 檔案**中
    * 易於部署和分享
### 🚀 **如何啟動**
* 這是一個**單一 HTML 檔案**，**不需要**任何複雜的設定或伺服器
* 將 **lens\_simulator.html** 檔案下載到您的電腦
* 使用任何**現代網頁瀏覽器**（如 **Chrome**, **Firefox**, **Edge**）開啟該檔案
* 即可開始模擬
### ⚙️ **技術棧**
* **HTML**: **結構**與**介面**
* **Tailwind CSS**: 負責**美觀**和**響應式設計**
* **JavaScript** (**Canvas API**): 負責所有**物理計算**和**圖形繪製**
