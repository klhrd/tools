# FluxTable

**FluxTable** 是一款專為開發者與數據工作者設計的「工業級」Markdown 表格編輯器。它結合了試算表的直觀操作與 Markdown 的純文字優勢，支援矩陣旋轉、轉置等進階功能，讓表格編輯不再痛苦。


## 🛠 核心功能

- **視覺化網格編輯**：像使用 Excel 一樣直觀地新增、刪除、拖曳調整行列。
- **Markdown 即時渲染**：在單元格內支援加粗、斜體、刪除線與程式碼格式。
- **網址/本地存儲**：自動同步編輯狀態至 `localStorage`，確保資料不因重新整理而遺失。
- **雙向同步機制**：
    - **視覺網格 (Visual Grid)**：直接編輯內容。
    - **原始碼編輯 (Code Editor)**：同步產出 Markdown 或 CSV 代碼。
- **進階矩陣運算**：
    - **轉置 (Transpose)**：行列對調運算 ($A^T$)。
    - **旋轉 (Rotate)**：支援逆時針 90 度旋轉表格。
- **智慧寬度控制**：支援「自動寬度 (Auto Width)」模式或手動拖曳欄位邊界。

## ⌨️ 快捷操作

- **撤銷/重做**：`Ctrl+Z` / `Ctrl+Y`
- **文字格式化**：選取文字後按下快捷鍵或工具列按鈕。
- **智慧刪除**：根據目前的焦點自動判定刪除列或刪除欄。
- **批次欄位操作 (Col Ops)**：支援全欄大寫轉換、添加括號包裹或一鍵清空欄位。

## 🚀 快速上手

1.  **環境**：這是一個單一 HTML 檔案，無需安裝，開啟瀏覽器即可運作。
2.  **匯入**：在右側代碼區切換至 `MARKDOWN` 或 `CSV` 模式並貼上內容，網格將自動解析。
3.  **調整**：使用頂部工具列進行行列增減、對齊（左/中/右）或執行矩陣變換。
4.  **匯出**：點擊 `COPY` 複製到剪貼簿，或點擊 `SAVE` 下載為 `.md` 或 `.csv` 檔案。

## ⚙️ 技術規格

- **字體**：JetBrains Mono (工業風等寬字體)
- **配色**：極黑工業主題 (Dark Industrial Theme)
- **引擎**：原生 JavaScript 表格引擎，支援複雜的 Markdown 語法解析。
