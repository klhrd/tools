# 🌑 [Moon Phase & Orbit Simulator (月相與食相原理模擬器)](https://klhrd.github.io/tools/lunar-phase-simulator/v0.1.9.html)

一個基於 WebGL 的互動式 3D 天文模擬工具。透過「上帝視角（軌道俯視）」與「地球視角（地面觀測）」的同步對照，直觀地展示月相變化的成因、月球軌道傾角以及日/月食的幾何原理。

## ✨ 核心特色 (Features)

### 🔭 雙視角同步顯示 (Dual-View Visualization)
* **軌道俯視圖 (Orbit View)**：從宇宙上方觀察地月系統，清楚看見太陽光方向、地球陰影與月球公轉軌道。
* **地球觀測視角 (Earth View)**：模擬從地球中心看向月球的真實景象，配合 **東(E)/西(W)** 方位標示，讓您理解為何「上弦月亮西邊，下弦月亮東邊」。

### 📐 真實天文物理模擬 (Physics & Astronomy)
* **軌道傾角 (Orbital Inclination)**：模擬月球軌道與黃道面約 **5.14°** 的夾角。這解釋了為什麼不是每個月圓之夜都會發生月食。
* **地球本影 (Earth's Umbra)**：視覺化顯示地球背後的紅色本影錐。當開啟此功能且月球穿過紅錐時，即代表發生**月食**。
* **月相週期**：從農曆初一（朔）到三十（晦）的完整週期模擬。

### 🖥️ 現代化介面與數據 (UI & Data)
* **即時數據儀表板**：顯示農曆日期、月相名稱、月出/月沒時間估算、以及太陽-地球-月球的相位夾角 (Phase Angle)。
* **智慧時間軸**：拖曳滑桿即可快速穿越時間，並具備關鍵月相（朔、望、弦）的磁吸定位功能。
* **自動播放**：一鍵觀看完整的月相盈虧循環。

### 🎨 高畫質渲染 (High-Fidelity Rendering)
* **極致銳利大氣層**：使用自定義 Shader 繪製的硬邊大氣層，消除模糊感，還原從太空觀測地球的清脆質感。
* **高解析度紋理**：支援視網膜螢幕 (Retina Display) 原生解析度，配合各向異性過濾 (Anisotropic Filtering)，展現清晰的月表坑洞。
* **沉浸式宇宙環境**：包含銀河系帶、動態星空與體積光塵效果。

## 🚀 如何使用 (How to Use)

1.  **開啟網頁**：直接瀏覽 `index.html` (或 GitHub Pages 連結)。
2.  **基本操作**：
    * 拖動下方的**時間滑桿**改變農曆日期。
    * 點擊 **播放 ▶** 按鈕自動演示。
    * 在 3D 場景中**按住左鍵拖曳**可旋轉視角，**滾輪**可縮放。
3.  **設定選單 (右上方齒輪)**：
    * **地球視角**：開啟/關閉右下角的子母畫面。
    * **全域光效**：開啟/關閉太陽光塵粒子。
    * **軌道傾角**：切換「平面軌道」或「真實 5° 傾角」。
    * **地球本影**：顯示/隱藏紅色的地球陰影錐（用於觀察月食原理）。

## 🛠️ 技術棧 (Tech Stack)

* **Three.js**: 核心 3D 引擎。
* **Tailwind CSS**: UI 樣式設計。
* **Glassmorphism**: 玻璃擬態介面風格。
* **Vanilla JS**: 純原生 JavaScript 邏輯，單一 HTML 檔案即可運作。
