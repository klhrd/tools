# finger-lottery

一個以手機為優先的多點觸控指尖抽籤工具，可即時抽出贏家。大家把手指放在螢幕上，按住幾秒後，會隨機選出一位，並以流暢的液體填充動畫揭曉結果。

## 功能特色

- **多點觸控支援**：同時支援2-10根手指。每根手指都會獲得獨特的高級感顏色。
- **高級配色方案**：內建10種精選顏色，如珊瑚紅、冰晶藍、薰衣草。可切換顯示顏色名稱或 P1、P2 標籤。
- **液體填充效果**：選出贏家時，螢幕會從觸碰點開始平滑地被該顏色填滿。
- **聲音回饋**：使用 Web Audio API 生成按壓、倒數、獲勝音效。可自由開關。
- **紙屑慶祝動畫**：選出贏家時可選擇噴發五彩紙屑特效。
- **可分享設定**：所有設定都會編碼到網址的 hash 中。複製連結分享給他人，即可還原完全相同的設定。
- **極簡模式**：隱藏所有介面文字，獲得純粹的視覺體驗。

## 使用方式

1. 在手機上打開頁面。
2. 讓2人以上將手指放在螢幕上。
3. 按住直到倒數結束。
4. 贏家的手指會放大，螢幕被其顏色填滿，並顯示名稱。

## 設定項目

點擊右上角漢堡選單開啟側邊面板：

- **計時器秒數**：調整按住後的抽籤時間，1秒至10秒可調。
- **新成員加入重置**：倒數中若有新手指加入則重新計時。
- **聲音回饋**：開關 Web Audio 音效。
- **紙屑特效**：開關獲勝時的紙屑動畫。
- **使用顏色名稱**：用高級顏色名稱取代 P1、P2。
- **顯示資訊面板**：顯示/隱藏上方狀態卡片。
- **極簡模式**：隱藏所有文字，保持畫面乾淨。

## 技術棧

- 原生 HTML、CSS、JavaScript
- Canvas API 渲染觸控與特效
- Web Audio API 合成音效
- 無任何依賴，載入後可離線使用

## 本地運行

直接在現代手機瀏覽器中打開 `index.html` 即可。  
建議在支援多點觸控的手機上使用，體驗最佳。
